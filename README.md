# Islo TypeScript SDK

Typed TypeScript SDK for the [Islo](https://islo.dev) sandbox platform. Create, manage, and execute code in isolated VM-based sandboxes.

## Installation

```bash
npm install @islo-labs/sdk
```

## Quick Start

```typescript
import { Islo } from "@islo-labs/sdk";

// Automatically reads ISLO_API_KEY from environment
const client = new Islo();

// Create a sandbox
const sandbox = await client.sandboxes.createSandbox({
    name: "my-sandbox",
    image: "ubuntu:22.04",
    vcpus: 2,
    memoryMb: 4096,
});

// Execute a command
const result = await client.sandboxes.execInSandbox({
    sandboxName: sandbox.name,
    command: ["echo", "hello world"],
});
console.log(result.exitCode);

// Clean up
await client.sandboxes.deleteSandbox({ sandboxName: sandbox.name });
```

## Authentication

### Environment variable (recommended)

```bash
export ISLO_API_KEY="your-api-key"
```

```typescript
const client = new Islo(); // Picks up ISLO_API_KEY automatically
```

### Explicit API key

```typescript
const client = new Islo({ apiKey: "your-api-key" });
```

### Custom token provider

```typescript
import { IsloClient } from "@islo-labs/sdk";

const client = new IsloClient({
    environment: "https://api.islo.dev",
    token: async () => fetchTokenFromYourAuthService(),
});
```

## Configuration

| Environment Variable | Description | Default |
|---------------------|-------------|---------|
| `ISLO_API_KEY` | API key exchanged for a short-lived JWT | — |
| `ISLO_BASE_URL` | API base URL | `https://api.islo.dev` |

## Development

```bash
# Install typecheck deps
npm install --no-save typescript@~5.6 @types/node@~20

# Typecheck
npx tsc --noEmit
```

### Regenerating the SDK

This SDK is generated by [Fern](https://buildwithfern.com/) from the OpenAPI spec maintained in [`islo-web-api`](https://github.com/islo-labs/islo-web-api). Fern runs from that repo and pushes the generated code here.

Custom code in `Islo.ts`, `index.ts`, `tsconfig.json`, and `custom/` is preserved across regenerations via `.fernignore`.
